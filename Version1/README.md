# Patience

2 inch Rocket
  Approximately:
                  400m max altitude
                  1 kg mass
                  
    
To start the rocket:
	open a terminal
		if it doesn't run automatically,
	type "cd Patience"
	type "python Patience.py"

	Open arduino IDE on RPI Ground Station
	Upload to arduino due via programming port DueSlv.ino
		Serial Commands:
			1:  Start program
			2:  Ignite rocket
			3:  Emergency Deployment/ Deployment
			4:  Reset
